# Pathfinding Research Through Hospitals

This initiative will study how to find the fastest route to hospitals from various points throughout a city using pathing and pathfinding algorithms.

## Algorithms Used for Pathfinding:
- Dijkstra's Algorithm
- A* Algorithm

## Features & Components:
- City model (graph)
- Path reconstructor (for creating and visualizing paths)
- Performance comparison (i.e. how long it takes, how much memory is used, how many nodes have been traversed)
